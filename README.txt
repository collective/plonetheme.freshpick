Introduction
============

An installable Diazo theme for Plone 4.1.

See: https://github.com/collective/plonetheme.unilluminated for more
information.

Installation
------------

Download the zip file and import it from the Diazo theme control panel:
http://plone.org/products/plonetheme.freshpick/releases/0.1.0/freshpick.zip.
